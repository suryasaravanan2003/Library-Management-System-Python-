The Library Manager is a command-line Python application used to maintain library book records and user transactions. It supports adding books, issuing and returning books, and tracking availability. All data is saved in JSON files, ensuring persistence without requiring a database.

Features:

Add books with title, author, ID

View complete book list

Search books by name or ID

Issue a book to a user

Return a book

Track available vs issued status

Automatic JSON storage management

Technologies Used:

Python 3

JSON for storing book data

File handling

Simple CLI menu system

Use Case:
Ideal for learning structured data management, file processing, and creating real-world mini-applications in Python.
